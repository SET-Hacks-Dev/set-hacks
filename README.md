### Running locally

First, clone the repo:

```
$ git clone https://github.com/malav-mehta/sethacks.ca.git
$ cd sethacks.ca
```

Then install the dependencies and get it running:

```
$ yarn install
$ yarn start
```

Finally, open `localhost:3000` in your browser.
