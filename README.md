## Running locally

First, clone the repo:

```
$ git clone https://github.com/malav-mehta/set-hacks.git
$ cd set-hacks
```

Then install the dependencies and get it running:

```
$ yarn install
$ yarn start
```

Finally, open [localhost:3000](http://localhost:3000) in your browser.
